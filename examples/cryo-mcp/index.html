What is Payflow?

Payflow is an agentic commerce library designed to empower AI agents to pay for MCP tools in real-time with stablecoin-based micropayments. Users can seamlessly integrate the Payflow-enabled demo MCP server powered by Cryo into Claude or Cursor to access its paid tools and turn them into powerful Ethereum data analysts. Check out some examples below.

On-chain micropayments

Payflow adopts the emerging x402 standard for agent-first commerce. This enables agents to make micropayments for individual data requests, offering a flexible, pay-per-query model. The payments are settled on Base and are paid in USDC.

Setup Required: Set your PRIVATE_KEY in the payflow configuration below. This wallet will be used to pay for cryo queries ({{PRICE}} USDC per query). It should be a wallet on Base with some USDC.
